# CampusFlow Frontend Client

This directory is designated for the frontend client of the CampusFlow student productivity platform.

## Proposed Tech Stack
* **Framework:** React.js (Vite)
* **Styling:** Vanilla CSS
* **Build Tool:** Vite
* **State Management:** React Context API or Redux Toolkit
* **API Client:** Axios (for connecting to the server backend and the Python RAG microservice)
